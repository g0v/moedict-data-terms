moedict-data-terms
==================

開放語料庫專案

### 雙語詞彙資料庫 (公告詞彙，機關建議詞彙) `public-terms`

 * Source: http://terms.naer.edu.tw/downloadlist/

#### How to start

```
cd public-terms
make download
```

 * Simple parser
```
./parser.py < data/中央機關主管職稱.xls
```
